# 🏠 House Price Prediction

This project predicts house prices using Machine Learning based on features like area, bedrooms, and bathrooms.

---

## 📊 Features Used
- Area (sq ft)
- Number of Bedrooms
- Number of Bathrooms
- location score
- age

---

## ⚙️ Technologies Used
- Python
- Pandas
- scikit-learn

---

## 🚀 How It Works
1. Data is loaded from CSV file
2. Features and target are separated
3. Linear Regression model is trained
4. Model predicts house prices

---

## ▶️ How to Run

1. Install required libraries:
pip install pandas scikit-learn

2. Run the program:
python main.py
